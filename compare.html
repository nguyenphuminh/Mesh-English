Câu so sánh

Công thức
• S1 + am/are/is + more + adj + than + S2
• S1 + V + more + adv + than + S2 + trợ động từ
--> Dùng để so sánh hơn

• S1 + am/are/is + as + adj + as + S2
• S1 + V + as + adv + as + S2 + trợ động từ
--> Dùng để so sánh bằng

• S1 + am/are/is + the most + adj
• S1 + V + the most + adv
--> Dùng để so sánh nhất
